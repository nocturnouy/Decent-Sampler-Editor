How it Works

The Decent Samples plugin is a FREE sample player plugin that allows you to play sample libraries in the DecentSampler format (files with extensions: dspreset and dslibrary). You can download the plugin, support it development and learn about all its features at decentsamples.com

Decent Sampler Editor is a project in development, many functionalities might not be implemented yet and will be added in the future

For now you can create and export your preset.dspreset file ready to use, you just need to make sure your samples are in the right directory.

You can download a boilerplate preset here, extract it, replace the preset.dspreset file with the one you can get from the editor and add your samples to the /samples folder

More detailed instructions on how to use the editor will be coming soon
